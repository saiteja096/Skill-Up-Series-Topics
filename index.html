Technologies Covered in Training

Git

Git is an open source virtual control system.Git was developed by Linux

GitHub

Github provides Source code management functionality of Git.You can have a Github account and host your projects.

MySQL

MySQL can be used to create database for storing and manipulating data.In MySQL data is stored in the form of tables.

Django

Django is python web framework.It is used to help you to develop websites faster and easier.

Flask

Flask is a micro web framework in python.It is used for developing web applications

Android Studio

Android Studio is Official IDE for app development.It provides tools required for building an app.
